How Does Javascript Work?

  • জাভাস্ক্রিপ্ট একটি interpreted language, এটি compiled language নয় এবং জাভাস্ক্রিপ্ট language একটি ক্লায়েন্ট-সাইড স্ক্রিপ্টিং এটি লাইন দ্বারা লাইন পদ্ধতিতে কাজ করা হয়|
  • আজকাল বেশিরভাগ আধুনিক ওয়েব ব্রাউজার জাভাস্ক্রিপ্ট সমর্থন করে এবং তাদের জাভাস্ক্রিপ্ট ইঞ্জিন রয়েছে।
  • Google Chrome এর নিজস্ব জাভাস্ক্রিপ্ট ইঞ্জিন রয়েছে যার নাম V8।
  • JavaScript ইঞ্জিন যা JavaScript কোড চালায়।
  • প্রথমে ওয়েব পেজ তৈরি করার জন্য এইচটিএমএল এবং সিএসএসকে DOM দ্বারা একত্রিত করা হয়।
  • তারপর, ব্রাউজারের জাভাস্ক্রিপ্ট ইঞ্জিন জাভাস্ক্রিপ্ট ফাইল এবং ইনলাইন কোড লোড করে কিন্তু কোডটি দ্রুত চালায় না।
  • এটি এইচটিএমএল এবং সিএসএস লোডিং শেষ হওয়ার জন্য অপেক্ষা করে জাভাস্ক্রিপ্টের কোন compilation পদক্ষেপ নেই।
  • ব্রাউজারে একজন interpreter জাভাস্ক্রিপ্ট কোড পড়ে, প্রতিটি লাইন ব্যাখ্যা করে এবং এটি চালায়।

How Does Even Loop Do?

  • ইভেন্ট লুপ হল এমন একটি প্রক্রিয়া যা টাস্ক কিউ থেকে কল স্ট্যাকে কলব্যাক ঠেলে দেওয়ার আগে কল স্ট্যাক পরিষ্কার হওয়ার জন্য অপেক্ষা করে৷
  • যখনই একটি async ফাংশন কল করা হয়, এটি একটি ব্রাউজার API এ পাঠানো হয়।
  • এগুলি ব্রাউজারে অন্তর্নির্মিত API, কল স্ট্যাক থেকে প্রাপ্ত কমান্ডের উপর ভিত্তি করে, API তার নিজস্ব একক-থ্রেডেড অপারেশন শুরু করে।
  • SetTimeout পদ্ধতি এটির উদাহরণ, যখন একটি সেটটাইমআউট অপারেশন স্ট্যাকের মধ্যে প্রক্রিয়া করা হয়|
  • তখন এটি সংশ্লিষ্ট API-এ পাঠানো হয় যা প্রক্রিয়াকরণের জন্য এই ক্রিয়াকলাপটিকে ফেরত পাঠানোর জন্য নির্দিষ্ট সময় পর্যন্ত অপেক্ষা করে ।
  • জাভাস্ক্রিপ্টে অ্যাসিঙ্ক অপারেশন চালানোর জন্য আমাদের একটি চক্রীয় সিস্টেম রয়েছে কিন্তু ব্রাউজার API গুলি আলাদা থ্রেড হিসাবে কাজ করে ইভেন্ট লুপ এই প্রক্রিয়াটিকে সহজতর করে।
  • এটি ক্রমাগত চেক করে যে কল স্ট্যাকটি খালি কিনা। এটি খালি থাকলে, ইভেন্ট সারি থেকে নতুন ফাংশন যোগ করা হয়।
  • যদি এটি না হয়, তাহলে current ফাংশন কল হবে।
